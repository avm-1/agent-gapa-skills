# Память проекта: 20_Agent_skills

Глобальная память для кросс-проектных паттернов, антипаттернов и выводов.
Пополняется через рефлексию навыка `gapa-self-learning`.

## Структура проекта

- `.agents/skills/gapa-self-learning/` — навык самообучения
  - `SKILL.md` — инструкция по рефлексии и извлечению паттернов
  - `references/reflection-template.md` — шаблон для анализа логов
  - `references/anti-pattern-log.md` — лог ошибок этого навыка

## Правила ведения памяти

- Добавлять новые записи в начало файла
- Формат антипаттернов — см. `gapa-self-learning/SKILL.md`
- Формат успешных паттернов — см. `gapa-self-learning/SKILL.md`
- Избегать дублирования: перед записью проверить, нет ли уже похожего паттерна
