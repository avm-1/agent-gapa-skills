## Успешный паттерн: Поиск файлов по маске

**Контекст**: Задачи, требующие найти все файлы определённого типа в директории.
**Проблема**: Команда `ls` не ищет во вложенных директориях и может пропустить файлы.
**Решение**: Всегда использовать `find . -type f -name "*.РАСШИРЕНИЕ"` для рекурсивного поиска.
**Проверка**: Вывести список найденных файлов перед дальнейшей обработкой.
**Пример**:
```bash
find . -type f -name "*.txt"
```

## Паттерн проекта: Подсчёт строк в файлах

**Контекст**: Подсчёт количества строк в текстовых файлах.
**Правило**: Всегда использовать `awk 'END {print NR}'` вместо `wc -l`.
**Причина**: `wc -l` не считает последнюю строку, если в файле нет завершающего перевода строки (`\n`). `awk 'END {print NR}'` корректно обрабатывает все строки независимо от наличия финального `\n`.
**Пример**:
```bash
awk 'END {print NR}' file.txt
```
