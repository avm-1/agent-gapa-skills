# AGENTS.md — Инструкции для агентов

## О проекте

Проект для развития самообучающихся агентских навыков, направленных на итеративное улучшение,
запоминание ошибок и их предотвращение при работе с кодом и инфраструктурой.

## Ключевые навыки

- `gapa-self-learning` — рефлексия, извлечение алгоритмов, запись антипаттернов
  - Триггер: завершение таска, 10+ tool calls, успех с нетривиальным решением, ошибка
  - Действие: провести рефлексию по шаблону, зафиксировать паттерны

## Структура

- `.agents/skills/` — project-level навыки
- `MEMORY.md` — глобальная память проекта (антипаттерны, паттерны)
- `.claude/settings.json` — настройки среды

## Конвенции

- Все новые skills создавать в `.agents/skills/<skill-name>/`
- Каждый skill содержит `SKILL.md` (обязательно) и опциональные `scripts/`, `references/`, `assets/`
- MEMORY.md пополнять только через осознанную рефлексию, не по рутине
